# ois-control
The data and codes are for the following paper:

P. Zhao, R. Nagamune and M. Chiao, \Multiple parameter-dependent robust control of miniaturized optical
image stabilizers,", Control Engineering Practice, provisionally accepted in Feb 2018.

The latest version can be found at https://github.com/boranzhao/ois-control. Please send inquiries and comments to boranzhao9@gmail.com or panzhao@mech.ubc.ca.
